Блок display

Class Menu - 
содержит основные поля и позволяет сделать выборку товаров по описанию
Class RegisteredUser - 
отображает вводимую пользователем информацию для регистрации и входа в личный кабинет
Сlass PersonalMenu - 
отображает дополнительный функционал для регистрированных пользователей. Использует Class Menu


Блок business logic

Class selectMenu - 
определяет статус регистрации и в зависипости от него запускает Class Menu или Сlass PersonalMenu
Class RegistrationForm - 
Сравнивает данные, введенные пользователем в форме (Class RegisteredUser) с базой данных (Class User и Class UsersRepo) и: подтверждает регистрацию, создает нового пользователя, и записывает в базу данных; или подтверждает вход в личный кабинет и запускает selectMenu.runPersonalMenu
Class ProductList - 
содержит 2 поля - список товаров и данные о зарегистрированном пользователе. делает выборку товаров по умолчанию (для незарегистрированных пользователей) или выборку по предпочтениям для зарегистрированных.
Class Payment - 
содержит информацию о зарегистрированном пользователе и получает положительный или отрицательный отклик от банка об оплате. при успешной оплате убирает товар из корзины (Class BasketUserRepo) и кладет в совершенные покупки (Class PaidProductsUserRepo)

Блок data

Class User - 
содержит основные поля
Class UsersRepo - 
формирует список пользователей, позволяет добавлять пользователей
Class Product - 
содержит основные поля, ипсользует Class DescrProd для описания товара
Class ProductsRepo - 
формирует список товаров
Class PaidProductsUserRepo - 
формирует список оплаченных товаров
Class BasketUserRepo - 
база данных корзины пользователя
